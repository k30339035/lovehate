# 변경 이력

## 2026년

### 2026년 1월
- 게임 플랫폼으로 전환
- 월 2,900원 구독제 도입
- 로봇 조종 게임 출시 예정 공지
- 네비게이션 게임 출시 예정 공지
- Vercel 자동 배포 설정 완료

### 주요 변경사항
- 홈페이지를 게임 플랫폼으로 재구성
- 글씨 드론 게임, 로봇 조종 게임, 네비게이션 게임 통합
- Supabase 데이터베이스 연동
- Stripe 결제 시스템 통합
- 성능 최적화 (동적 임포트, 코드 스플리팅)

### 2024년 초기
- 프로젝트 시작
- 드론 글씨 쓰기 앱 개발
